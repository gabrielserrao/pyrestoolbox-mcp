### üöÄ Quick Start

```bash
# Install UV
curl -LsSf https://astral.sh/uv/install.sh | sh

# Clone and setup
git clone https://github.com/YOUR_ORG/pyrestoolbox-mcp.git
cd pyrestoolbox-mcp
make uv-install

# Verify (all 47 tools should pass)
uv run python test_tools.py

# Run server
make uv-server
```
